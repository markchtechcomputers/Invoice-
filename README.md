# Invoice Scanner & Management System

A professional invoice scanning and management system for businesses to process supplier invoices.

## Features

- **Camera Scanner**: Capture invoices using device camera
- **OCR Processing**: Extract invoice number, LPO number, supplier name
- **PDF Storage**: Store and manage PDF invoices
- **Advanced Search**: Search by invoice #, LPO #, supplier, date
- **Dashboard**: Real-time statistics and insights
- **Export**: Export data to CSV
